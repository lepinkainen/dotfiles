description: Analyse and fix a GitHub issue
---

Please analyze and fix the GitHub issue: $ARGUMENTS

Follow these steps:

1. Use `gh issue view` to get the issue details
2. Understand the problem described in the issue
3. Search the codebase for relevant files
4. Create a new branch for the fix
5. Implement the necessary changes to fix the issue
6. Write and run tests to verify the fix
7. The code MUST build, pass linting, and type checking
8. Create a descriptive commit message
9. Push to the created branch and create a PR linked to the issue

Remember to use the GitHub CLI (`gh`) for all GitHub-related tasks.
